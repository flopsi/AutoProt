# LFQ Proteomics Analysis Platform

Thermo Fisher Scientific - Mass Spectrometry Data Analysis

## Features
- Protein/Peptide data upload (CSV/TSV/Excel)
- Automatic species detection (HUMAN, ECOLI, YEAST)
- LFQ Bench workflow (A vs B comparison)
- Interactive visualizations

## Deployment
Deploy to Streamlit Cloud via GitHub.

## Usage
1. Navigate to Protein Upload
2. Upload data or use demo
3. Follow guided workflow
4. View interactive results
